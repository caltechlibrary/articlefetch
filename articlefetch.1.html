NAME

articlefetch

SYNOPSIS

articlefetch OPTIONS HOSTNAME QUERY_STRING

DESCRIPTION

Take the HOSTNAME and QUERY_STRING values, retrieve the results from RDM and then using the results retrieve the PDFs.

OPTIONS

-help
display this help page
-version
display version info
-license
display license

EXAMPLE

articlefetch authors.library.caltech.edu "Grubbs, Robert"
